Módulo E2E Karate

Este módulo contiene las pruebas end-to-end escritas con Karate.

  • Ejecución local: mvn -q -f e2e/karate/pom.xml test
  • Estructura de tests en src/test/resources/tests
  • Runner JUnit5 en src/test/java

Más detalles sobre convenciones, hooks y buenas prácticas se documentarán aquí.
