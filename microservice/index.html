Módulo Microservicio (code)

Este módulo opcional contiene un microservicio Spring Boot de ejemplo con un endpoint /products.

  • Arranque local: mvn -q -f code/pom.xml spring-boot:run
  • Puerto configurable con APP_PORT o server.port.
  • Genera dos productos de ejemplo.

Se usa para apoyar pruebas E2E que apunten a un servicio real.
